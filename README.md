# 小组报数日记

这是一个独立的每日报数、达标审核和小组总览工具。

## 启动

推荐双击 `run_software_app.bat`，用桌面软件模式打开。

第一次运行会自动安装 Electron 桌面运行时，之后就是普通软件窗口。

打开软件后需要输入应用密码，默认是 `999`。管理员在“权限”里修改密码并保存配置后，下次打开会使用新密码。

如果只是临时查看，也可以双击 `run_web_app.bat`，或直接用 Chrome / Edge 打开 `index.html`。

`app.ts` 是 TypeScript 源码，`app.js` 是界面运行版本。

## 日常使用

在“今日”里直接填写每个项目的数字输入框，例如视频、音频、字幕、图片分别填数量即可。

系统会按管理员面板里的项目换算系数统计原始总数、换算总数，并自动判断是否达标。

## 整体预览

“整体预览”会显示当天所有成员的达标情况：

- 达标人数、未达标人数、完成率
- 每个成员的换算数、定额、进度条
- 小组历史记录总览

## 管理员

“管理员”面板可以统一调整：

- 默认定额
- 项目数量/项目名称/换算系数
- 成员名单
- 每个成员的个人定额
- 成员分组和排序

这些配置都保存在同一个数据 JSON 里。选择共享文件后，配置会和日报记录一起写入 Google Drive 同步目录。

管理员修改项目、默认定额、个人定额、成员名单后，可以点“保存配置”。配置会写入共享数据文件，其他成员的软件或云端页面轮询到更新后会自动刷新。

“成员与分组”里可以新增成员、新增分组、给成员选择分组，并用上移/下移调整成员顺序。整体预览的分析范围支持团队、分组、个人。

“分组项目配置”里可以按组勾选需要填写的项目。成员默认继承自己所在组的项目配置；今日报数默认只显示该成员所在组选中的项目，也可以点“全部显示”临时展开全部项目。

进入“管理员”需要输入管理员密码，默认是 `999`。可以在“权限”里修改密码，保存配置时会同步到云端数据，并在保存前自动备份。

## 效率分析

“整体预览”里包含：

- 今日个人差额和团队总差额
- 可选择任意查看日期，默认使用本地系统当天日期
- 个人/团队近 7 天、近 30 天完成量柱状图
- 项目数量汇总图
- 对比上周期、昨天、前天、上月同期、本周累计的完成量差距
- 独立“对比栏目”：当前 N 天 vs 前 N 天，支持 2 天、7 天、30 天和自定义天数
- 折线趋势、矩形树图、小方块进度预览
- 按日期范围查看个人或团队明细表

成员填写时会先保存本地草稿，点“提交”后才同步到云端，并根据管理员设置弹出自动审核反馈。

点击“表格备份”会生成表格备份。软件模式下会写入 Google Drive 同步文件夹的 `sheet_backups` 子目录。

管理员默认开启“提交后自动更新 3 个月表格”：成员提交后会自动生成/更新一个最近 3 个月的 `.xls` 工作簿，包含“总览”“全部记录”和每个成员单独页面，可用 Excel 或 Google 表格导入打开。云端数据文件本身不会自动清理。

“本地清理”默认关闭。管理员打开后才会清理本地小备份，并保留周备份、月备份、配置、恢复等重要备份。

## Google Drive 云同步

软件模式里点击“云端文件夹”，选择 Google Drive 桌面同步目录里的文件夹，例如：

```text
G:\My Drive\小组报数
```

应用会自动在这个文件夹里创建或使用 `report_data.json`。其他成员选择同一个共享文件夹即可同步。软件会定时读取共享文件变化，并把本地修改写回该文件。

软件模式会记住这个文件夹路径，下次打开会自动继续同步。

注意：多人同时修改同一条记录时，最后保存的人会覆盖这条记录；建议每个人只编辑自己的成员记录。

## 本地备份

应用会保留本地浏览器副本，并在保存、导入、切换云端文件、云端刷新前创建备份。

备份会自动清理 90 天以前的内容，避免长期堆积。可以在“管理员”里的“同步与备份”查看和恢复。
